- Descrição:
  -O projeto consiste em um gerenciaor de festas, onde há três tipos de login: 
    -Usuario, que pode ver e contratar festas;
    -Funcionario, que pode ver em quais festas irá trabalhar;
    -admin, que cuida de todo o gerenciamento do sistema, ver, contratar e demitir funcionarios, ver usuarios e suas festas contratadas, ver todas as festas e editar as mesmas.
- Divisão do trabalho:
  -Renan ficou responsável pelo servidor, estruturação base das views com ejs e banco de dados;
  -Isabel e João ficaram responsáveis pela estruturação final das views, bem como o css e a responsividade;
- Tecnologias:
  - Front-end: CSS, JavaScript e arquivos .ejs;
  - Back-end: express.js e Sequelize;
  - Banco de dados: MySQL
  - Segurança: Bcrypt;
- Como rodar:
  - Para rodar a aplicação, certifique-se de ter o MySQL intalado, altere a senha "aluno123" para a sua senha em src/config/bd.js, abra o terminal do VSCode ou o seu ambiente respectivo e digite "node server.js", acesse localhost:3000 no seu navegador e pront;
  - o login de admin é admin e a senha "admin123";
  - para logar como funcionario, veja seu nome de usuario e sua senha será sua função+123;
